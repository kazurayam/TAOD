      final class Index
extends java.lang.Object

      The "index" file is a list of files contained in the "Job" directory. The line contains a set of metadata about each file. Every lines of "index" is in the format as follows 1. ID as 40 HexDecimal characters 2. seperated by a TAB 3. FileType such as "png" = filename extension 4. seperated by a TAB 5. Metadata in List literal: "data" seperated by a comma, enclosed by [ and ] e.g, 6141b40cfe9e7340a483a3097c4f6ff5d20e04ea\tpng\t{"URL":"http://demoaut-mimic.kazurayam.com/","profile":"DevelopmentEnv"}
