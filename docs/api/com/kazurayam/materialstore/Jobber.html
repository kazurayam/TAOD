        • Material write(byte[] data, FileType fileType, Metadata metadata)

          This method writes the "byte[] data" into a File on disk. The path of the file will be ///objects/. And the "index" file will records MObjects; 1 line per 1 single MObject. An entry of "index" will be like: ¥t¥t The "index" entries are identified uniquely by the combination of + You can not write (create) 2 or more MObjects (=index enteries) with the same + combination. If you try to do write it, a MaterialstoreException will be raised. However, you can create 2 more more MObjects (=index entries) with different key with the same `.`. This means, you may possibly see such an index entries:
